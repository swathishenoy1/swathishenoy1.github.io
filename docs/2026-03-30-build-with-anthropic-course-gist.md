This article is about the Anthropic course "Build with Claude API". At a high level, I will outline the topics covered and the key learnings from this course.

## Topics
- Claude Models
- How to use Claude API
- System prompts
- Temperature
- Response Streaming
- Prompt Evaluation
- Prompt Engineering
- Tool use
- Retreival Augmented Generation (RAG) and Agentic Search
- Extended thinking, citations, prompt caching, Files API and code execution on Claude
- Model Context Protocal (MCP) - What it is, why we need it and how to build MCP Client and server
- Claude Code and computer use - simple walk through of how to use claude code 
- Workflows and Agents - what they are and when to use each

## Key Learnings
As a novice to the GenAI space, I had primarily experimented with tools like ChatGPT, Claude Code, and Codex to generate content and learn new concepts. However, I quickly realized that the quality of the output is heavily dependent on the prompts provided. I enrolled in this course out of curiosity to understand how these systems work under the hood.

The course helped me become familiar with common terminology and clarified many of the “how” and “why” behind LLMs. One key takeaway was viewing Claude as an effort multiplier—when used effectively with techniques such as prompt evaluation, system prompts, and structured prompt engineering, it can significantly improve outcomes. The more precise and well-structured the prompt, the better the results.

The course itself is well-designed, combining conceptual overviews with hands-on exercises. It strikes a good balance by providing sufficient depth without overwhelming beginners. Overall, it has given me a strong foundation to explore more advanced topics, including GenAI and Agentic AI, and even dive into research papers. And as a bonus—it’s completely free.
